# beijingPaymentBot

一个极简的Telegram支付指导机器人，提供支付宝和微信支付方式展示。只在私聊中工作，包含二维码图片和口令红包指导。

## 功能特性

- ✅ **支付指导服务**：提供支付宝和微信支付指导
- ✅ **二维码展示**：支持发送支付二维码图片
- ✅ **口令红包**：提供支付宝口令红包创建指导
- ✅ **私聊专用**：只在私聊中响应，不干扰群组消息
- ✅ **无数据库**：无需复杂配置，直接运行
- ✅ **极简设计**：只有一个命令，界面清晰

### 💰 支付服务

#### 支付宝支付
- 📱 **扫码支付**：展示支付宝二维码图片
- 🧧 **口令红包**：提供口令代码供用户领取
- 💬 **详细指导**：步骤清晰的支付说明

#### 微信支付
- 📱 **扫码支付**：展示微信二维码图片
- 💬 **详细指导**：步骤清晰的支付说明

#### 客服服务
- 📞 **微信联系**：提供店主微信号（xymh0923）
- 💬 **在线咨询**：随时联系客服获取帮助

## 环境要求

- Python 3.8+
- pip
- Telegram Bot Token（从 @BotFather 获取）
- 支付宝和微信二维码图片文件

## 安装步骤

1. 克隆或下载项目文件

2. 安装依赖：
```bash
pip install -r requirements.txt
```

3. 准备支付二维码图片：
   - 将支付宝二维码保存为 `alipay_qr.png`
   - 将微信二维码保存为 `wechat_qr.png`
   - 将图片文件放在项目根目录

4. 配置环境变量：
```bash
cp env_example .env
# 编辑 .env 文件，填写实际配置信息
```

5. 配置环境变量：
```bash
# 编辑 .env 文件，只需要配置Bot Token即可
TELEGRAM_BOT_TOKEN=your_bot_token_here       # 从 @BotFather 获取

# 其他配置都有默认值，无需修改
```


## 运行机器人

```bash
python main.py
```

机器人启动后，只在私聊中响应消息。

## Telegram Bot 配置

1. 创建机器人：
   - 在Telegram中找到 @BotFather
   - 发送 `/newbot` 创建新机器人
   - 获取Bot Token

2. 准备支付材料：
   - 支付宝二维码图片 (`alipay_qr.png`)
   - 微信二维码图片 (`wechat_qr.png`)
   - 支付宝口令红包代码（可选）

## 使用说明

### 基本操作

1. 用户在Telegram中私聊机器人
2. 发送 `/start` 查看支付方式
3. 点击按钮获取对应支付指导

### 支付方式

- 💚 **支付宝** - 显示二维码图片和口令红包指导
- 💙 **微信** - 显示二维码图片
- 🧧 **口令红包** - 显示创建步骤
